XOne 技术架构

个人生活/工作管理一体化平台 - 微内核 + 插件化架构

前端技术栈

  • • Next.js 14+ (App Router) - SSR/SPA 双模
  • • Shadcn UI + Ant Design - 高定制组件库
  • • Zustand - 轻量状态管理
  • • Tailwind CSS 4 - 四主题系统
  • • Recharts + ECharts - 数据可视化
  • • next-intl - 国际化支持

后端技术栈

  • • Python 3.12 + FastAPI - 异步原生
  • • Pydantic v2 - 类型安全
  • • Celery + Redis - 任务队列
  • • Meilisearch - 全文搜索引擎
  • • MinIO - S3 兼容对象存储
  • • Qdrant - 向量数据库 (RAG)

数据与基础设施

  • • PostgreSQL 16 - 主数据库 (强事务)
  • • MongoDB - 文档数据库 (灵活 Schema)
  • • Redis - 缓存 + 消息队列
  • • Nginx + Let's Encrypt - 反向代理
  • • Docker Compose - 容器编排
  • • 一键部署 - 10分钟启动

XOne v1.0 • 个人生活/工作管理一体化平台 • 微内核 + 插件化架构
